# Ground Station Planner

Plan and schedule satellite tracking sessions using TLE Parser and FullCalendar integration.

## Features

- Station configuration management
- Multi-satellite scheduling
- Conflict detection
- Coverage visualization
- CSV/iCal export
- Pass predictions

## Technology Stack

- Vue.js 3
- FullCalendar
- TLE Parser
- satellite.js

## Setup

```bash
cd demos/06-ground-station-planner
npm install
npm run dev
```

## TLE Parser Integration

Parses satellite TLEs to schedule ground station tracking sessions and detect scheduling conflicts.

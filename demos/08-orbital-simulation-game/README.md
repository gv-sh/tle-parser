# Orbital Simulation Game

Educational game teaching orbital mechanics using real SGP4 physics from TLE data.

## Features

- Orbital mechanics challenges
- Tutorial mode
- Scoring system
- Multiple levels
- Real SGP4 physics
- TLE-based satellite behavior

## Technology Stack

- Phaser.js
- TLE Parser
- satellite.js (SGP4)

## Setup

```bash
cd demos/08-orbital-simulation-game
npm install
npm run dev
```

# TLE File Manager

Comprehensive TLE file management tool with drag-drop upload, parsing, validation, search, and export capabilities.

## Features

- Drag-drop TLE file upload
- Parse and display TLEs in table format
- Search/filter functionality
- Checksum validation
- Export to JSON, CSV, XML formats
- Diff viewer for TLE changes
- Batch operations

## Technology Stack

- React
- TanStack Table
- TLE Parser

## Setup

```bash
cd demos/07-tle-file-manager
npm install
npm run dev
```

## TLE Parser Integration

Demonstrates TLE file parsing, validation, and export in multiple formats.

# Satellite Photography Planner

Plan ISS and satellite photography sessions using TLE Parser.

## Features

- ISS/satellite photography planning
- Camera settings calculator
- Pass timing
- Weather integration
- Equipment recommendations
- Photography tips

## Technology Stack

- Vue.js
- TLE Parser
- satellite.js

## Setup

```bash
cd demos/11-satellite-photography-planner
npm install
npm run dev
```

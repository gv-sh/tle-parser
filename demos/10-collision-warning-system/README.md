# Collision Warning System

Real-time satellite conjunction monitoring and collision risk analysis using TLE Parser.

## Features

- Conjunction monitoring
- Miss distance calculation
- Risk alerts dashboard
- Timeline visualization
- Report export
- Probability calculations

## Technology Stack

- React
- Chart.js
- TLE Parser
- satellite.js

## Setup

```bash
cd demos/10-collision-warning-system
npm install
npm run dev
```

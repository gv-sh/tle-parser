# Educational Astronomy Tool

Interactive 3D lessons about orbital mechanics using TLE Parser and Three.js.

## Features

- Interactive orbital lessons
- Parameter visualization
- Quiz mode
- Glossary
- Real satellite examples
- 3D visualizations

## Technology Stack

- React
- Three.js (@react-three/fiber)
- TLE Parser
- satellite.js

## Setup

```bash
cd demos/09-educational-astronomy
npm install
npm run dev
```

# Documentation Command

Create, improve, or review documentation.

## Instructions

You are invoking the **Documentation Agent**. Follow the system prompt at `.claude/agents/documentation.md`.

**Task:** Work on documentation as specified by the user.

**Steps:**
1. Read the agent system prompt at `.claude/agents/documentation.md`
2. Identify the documentation target:
   - API documentation
   - Usage guides
   - Examples
   - README updates
3. Review existing documentation for style consistency
4. Write or update documentation following:
   - Clear, accessible language
   - Code examples that are complete and runnable
   - Proper markdown formatting
   - Consistent terminology (see TLE glossary in agent prompt)
5. Verify technical accuracy against the source code

**Documentation types:**
- API Reference: Function signatures, parameters, return types
- Guides: Step-by-step tutorials
- Examples: Runnable code samples
- README: Project overview
